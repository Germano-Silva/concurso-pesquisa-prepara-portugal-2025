# Relatório de Viabilidade: O Perfil Sócio-profissional do Imigrante em Portugal

## Resumo Executivo

O conjunto de dados disponíveis (relatórios do INE de 2011) permite responder **parcialmente** às quatro perguntas de pesquisa. A análise revela **viabilidade alta** para duas perguntas (perfil educacional por setor e diferenças por nacionalidade) e **viabilidade média/baixa** para as demais devido a limitações temporais e de comparabilidade com a população nacional. Os dados são estáticos (2011) e concentram-se em indicadores agregados, exigindo complementação para tendências temporais e comparações setoriais detalhadas com a população portuguesa.

## Análise por Pergunta de Pesquisa

### Pergunta 1: Evolução temporal do nível de escolaridade

**Viabilidade: Baixa**  
- Dados disponíveis: Apenas 2011, sem série temporal  
- Lacunas: Faltam dados educacionais históricos  
- Recomendação: Buscar fontes como Pordata ou Eurostat para séries temporais

### Pergunta 2: Distribuição setorial vs. população nacional

**Viabilidade: Média**  
- Dados disponíveis: Profissões principais dos imigrantes  
- Lacunas: Falta detaill setorial dos portugueses  
- Recomendação: Cruzar com dados setoriais da população nacional

### Pergunta 3: Perfil educacional por setor

**Viabilidade: Alta**  
- Dados disponíveis: Nível educacional + profissão por nacionalidade  
- Análise possível: Cruzamento direta de variáveis  
- Exemplo: "No setor de construção, 65% dos imigrantes têm ensino secundário"

### Pergunta 4: Diferenças por nacionalidade

**Viabilidade: Alta**  
- Dados disponíveis: Nível educacional por nacionalidade  
- Análise possível: Comparação direta entre países  
- Exemplo: Contraste entre Portugal e Bangladesh

## Conclusões e Recomendações

### Pontos Fortes
- Dados educacionais e laborais detalhados para 2011
- Desagregação por nacionalidade e profissão
- Indicadores claros para análises descritivas

### Limitações
- **Falta de série temporal** para análise de evolução
- **Ausência de detalhe setorial** da população nacional
- **Dados estáticos** (2011) limitando análise dinâmica

### Recomendações
1. Para evolução temporal:  
   - Consultar relatórios do INE posteriores  
   - Analisar bases de dados educacionais históricas

2. Para comparação setorial:  
   - Usar dados setoriais do INE ou Pordata  
   - Considerar indicadores de empregabilidade por setor

3. Para análise avançada:  
   - Aplicar técnicas de visualização para mapear perfil educacional  
   - Realizar análise de correspondência entre educação e profissão

## Classificação Resumida

| Pergunta | Viabilidade |  
|----------|-------------|  
| Evolução educacional | Baixa |  
| Distribuição setorial | Média |  
| Perfil por setor | Alta |  
| Diferenças por nacionalidade | Alta |  

**Conclusão Final:** O tema de pesquisa é viável com ajustes metodológicos, principalmente para as Perguntas 1 e 2. Os dados de 2011 são robustos para análise cruzada, mas exigem fontes complementares para tendências temporais e comparações nacionais.
